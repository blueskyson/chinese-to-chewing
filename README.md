## 注音文轉換器

無聊做的，只有用新酷音在 Ubuntu 20.04 測試過

```
$ python c2c.py "哈囉，你好！"
ㄏㄚ ㄌㄨㄛ ，ㄋ一ˇㄏㄠˇ！
```
